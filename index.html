Introduction

JavaScript is a powerful scripting language used in web development. It runs in the browser and allows dynamic behavior on websites. JavaScript can be used for both front-end and back-end development. It's supported by all modern browsers. Learning JavaScript is essential for becoming a full stack developer.

  • Client-side scripting
  • High-level language
  • Interpreted and event-driven
  • Used with HTML and CSS
  • Rich ecosystem
console.log("Hello, JavaScript!");
Variables and Data Types

You can declare variables using let, const, and var. JavaScript supports various data types: string, number, boolean, object, null, undefined. Type coercion is automatic in some operations. Use typeof to check variable types.Best practice is to prefer let and const.

  • String: "Hello"
  • Number: 42
  • Boolean: true
  • Object: { key: value }
  • Undefined / Null
let name = "Davide";
Functions

Functions allow you to reuse code and structure your logic. You can define them with function declarations or expressions. Arrow functions are a concise syntax introduced in ES6. Functions can return values and accept parameters.Functions can be passed as arguments (first-class citizens).

  • function declaration
  • arrow function
  • callback function
  • anonymous function
  • IIFE
function greet(name) { return "Hello " + name; }
Loops

Loops let you run code multiple times based on conditions. JavaScript supports for, while, and do...while loops. You can use break and continue to control flow. Iterating over arrays is common with for...of and forEach. Be careful with infinite loops!

  • for
  • while
  • do...while
  • for...of
  • Array.forEach()
for(let i=0; i<5; i++){ console.log(i); }
DOM Manipulation

The DOM is a representation of the HTML structure as objects. You can access and modify elements with JavaScript. Common methods include getElementById, querySelector. You can change text, styles, and attributes dynamically. Event listeners allow interaction handling.

  • document.getElementById()
  • document.querySelector()
  • element.innerText
  • element.style
  • addEventListener()
document.getElementById("btn").addEventListener("click", handleClick);
